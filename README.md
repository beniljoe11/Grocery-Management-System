# Grocery Management System | Python Project

## Overview
This project is a simple Grocery Management System developed using Python. It allows users to add, remove, view grocery items, and calculate the total price through a command-line interface.

The project demonstrates core Python programming concepts such as functions, loops, dictionaries, conditional statements, and modular programming.

## Features
- Add grocery items with price
- Remove items from the grocery list
- Display all grocery items
- Calculate total price of items
- Interactive menu-driven program

## Technologies Used
- Python
- Dictionary Data Structure
- Functions
- Loops & Conditional Statements

## Example 

```bash
1. Add Items
2. Remove Items
3. View Items
4. Total Price
5. Exit
```

## Learning Outcomes
This project demonstrates:
- Python fundamentals
- Function-based programming
- Dictionary operations
- User input handling
- Modular programming using imports
- Basic backend logic implementation

## Project Purpose
This project was created to practice Python programming and build a simple command-line grocery management application.
